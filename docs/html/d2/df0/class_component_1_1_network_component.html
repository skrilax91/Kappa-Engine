Detailed Description

Network component.

This component is used to define if the entity is networked or not

If the entity is networked, it will be replicated on the network

If the entity is not networked, it will be only on the local machine

Parameters
clientPredictionIf the entity is predicted by the client
ignorePredictionIf the entity is predicted by the server
asAuthorityIf the entity is the authority of the prediction
